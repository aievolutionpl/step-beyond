# Install Step Beyond in ChatGPT

Paste `templates/chatgpt-core-instruction.txt` into Custom GPT Instructions,
Project Instructions, or at the start of a normal chat.

This produces prompt-only behavior. ChatGPT conversation or project memory may
provide context, but do not describe it as a Step Beyond audited store. External
publication and other consequential actions still require explicit permission.

Smoke test:

```text
Only correct the typo in the attached file, nothing else.
```

A conforming answer changes only the typo, provides no unsolicited proposal, and
states only verification actually performed.
